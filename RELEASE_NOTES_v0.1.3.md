# What's new in v0.1.3

## Features

- **Server-driven filtering, sorting, and search for flows** ([#104](https://github.com/zenml-io/kitaru-ui/pull/104)) — The flows table now queries the server for filtering, sorting, and search instead of doing it client-side. The URL is the single source of truth — filters and sort order persist across page refreshes and navigation. Flows are now sorted by latest execution by default.

**Full Changelog:** https://github.com/zenml-io/kitaru-ui/compare/v0.1.2...v0.1.3
